# Aura Photography — Installation Guide

## Requirements
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A local web server (recommended) or static hosting platform

## Quick Start (Local)

### Option 1: VS Code Live Server
1. Open the `photography-studio` folder in VS Code.
2. Install the "Live Server" extension.
3. Right-click `pages/index.html` and select "Open with Live Server".

### Option 2: Python HTTP Server
```bash
cd photography-studio
python -m http.server 8000
```
Open `http://localhost:8000/pages/index.html` in your browser.

### Option 3: Node.js (npx serve)
```bash
cd photography-studio
npx serve .
```
Navigate to `/pages/index.html`.

## Deployment

Upload the entire `photography-studio` folder to any static host:
- Netlify
- Vercel
- GitHub Pages
- AWS S3 + CloudFront

Set your site root to the `photography-studio` directory. Update URLs in
`sitemap.xml`, `robots.txt`, and JSON-LD in `pages/index.html` with your domain.

## Integrations to Configure

1. **Contact Form (Formspree)**
   - Sign up at https://formspree.io
   - Replace `YOUR_FORM_ID` in `pages/contact.html`

2. **Newsletter (Mailchimp)**
   - Create a signup form in Mailchimp
   - Replace placeholder action URLs in `pages/index.html` and `pages/coming-soon.html`

3. **Google Maps**
   - Generate an embed from Google Maps
   - Replace the iframe `src` in `pages/contact.html`

4. **Booking Calendar**
   - Add Calendly, Acuity, or similar widget in `pages/service-details.html`
   - Look for the `CALENDAR INTEGRATION` comment

## Images

Demo images load from Pexels CDN. For production:
1. Replace with your own photography.
2. Save as WebP in `assets/images/`.
3. Update `src` attributes in HTML files.

## File Structure
```
photography-studio/
├── assets/css/     — Stylesheets
├── assets/js/      — JavaScript + Lightbox2 plugin
├── assets/images/  — Your WebP images
├── pages/          — All HTML pages
├── documentation/  — Guides and credits
├── sitemap.xml
└── robots.txt
```
